# Overview

A command line tool for managing personal notes with various use cases.

# scheduler-app

A Kotlin note-taking app with various use cases including, quick note capture, which allows the user to add notes with a title, content, tags. The other use cases include organization and categorization, priority management, sorting and browsing, and persistent storage.
